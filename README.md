# 🇲🇽 Mexico-Lingo — Naturalización Mexicana

Duolingo-style flashcard app to study for the Mexican naturalization exam.

## 🌐 Use it / Install it

👉 **https://stajulian5.github.io/mexico-naturalizacion/**

Open that link in Chrome, Edge, or Safari and install it as an app:

- **Chrome / Edge (Mac/Windows):** click the install icon in the address bar, or open the **⋮ menu → Save and Share → Install Mexico-Lingo...**
- **Safari (Mac):** **File menu → Add to Dock**
- **iPhone / iPad (Safari):** tap **Share → Add to Home Screen**
- **Android (Chrome):** tap **⋮ menu → Install app**

## 📚 What's inside

- 439 questions across 11 categories (geography, history, civics, culture, etc.)
- Duolingo-style learning path with 9 chapters / 31 lessons
- XP, streaks, hearts, and study modes (all questions, missed only, simulated exam, etc.)
- Works offline once installed (PWA with service worker)
